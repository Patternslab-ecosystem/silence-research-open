# H01: φ-Proportioned Spacing Reduces Attentional Friction

**ID:** H01
**Status:** Active Investigation
**Opened:** 2026-Q1

## Hypothesis

UI layouts using φ-scaled spacing (8, 13, 21, 34, 55px) produce measurably
lower attentional friction scores compared to uniform 8px grid layouts,
as measured by Pattern Compliance Score over 5-minute observation sessions.

## Rationale

Golden ratio proportions appear throughout natural visual systems. If the
brain has an inherent processing affinity for φ-proportioned structures,
φ-scaled spacing should reduce the cognitive effort required to navigate
structured information.

## Measurement Protocol

1. Two cohorts observe identical content in φ-spaced vs. uniform-spaced layouts
2. SilenceEventV2 events collected with ATTENTION_SHIFT and FLOW_STATE types
3. calculatePCS() computed over 5-minute sessions
4. Statistical significance test: two-sample t-test, α = 0.05
